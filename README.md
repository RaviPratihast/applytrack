# ApplyTrack

ApplyTrack is a simple job application tracking tool built to manage and organize my job search.

The goal of this project is to keep track of applications, interview progress, follow-ups, and notes in one place, without over complicating the process.

This project is being built incrementally, starting with a clean React setup and gradually adding features with a focus on usability and clarity.

Tech stack:
- React
- JavaScript
- Vite
- Tailwind CSS
- shadcn/ui
